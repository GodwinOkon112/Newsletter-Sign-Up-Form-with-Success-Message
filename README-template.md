# Newsletter Sign-Up Form with Success Message

## Overview

This is a responsive newsletter sign-up form application that validates user input and displays a dynamic success message upon successful submission.

The project focuses on improving user experience through proper validation, controlled form submission, and dynamic data handling between pages.

### The Challenge

Users should be able to:

- Enter their email and submit the form
- See validation errors if:
  - The field is empty
  - The email format is invalid

- Successfully submit the form when valid
- Be redirected to a success page
- See their submitted email displayed on the success page

### What I Learned

This project helped reinforce several important frontend concepts:

- Form Validation Logic:
  How to properly validate user input using JavaScript before submission.

- Preventing Default Behavior:
  Using `event.preventDefault()` to stop unwanted page reloads.

- Regex for Email Validation:
  Implementing pattern matching to ensure correct email formats.

- Dynamic Page Navigation:
  Passing data between pages using query parameters.

- DOM Manipulation:
  Updating UI elements dynamically based on user input.
